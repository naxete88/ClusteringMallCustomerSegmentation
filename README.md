# ClusteringMallCustomerSegmentation
A clustering exercise on consumers in a shopping center using the K-Means algorithm.

Mall_customers is the initial data file, you can find it at the link:
https://www.kaggle.com/datasets/shwetabh123/mall-customers. 
Subsequently ClusteringMallCustomerSegmentationData is the file in Jupyter Notebook where the exercise is performed. 
The ClusteredMallCustomers file is the excel file with the assigned clusters. Finally, if you want to see an interactive graph in tableau you can do so at: 
https://public.tableau.com/app/profile/ignacio.macaya.bernad/viz/ClusteringMallConsumers/Dashboard1?publish=yes
